# 需求分析

UNO是一种著名的牌类游戏，于1971年由Merle Robbins发明，现由游戏公司Mattel生产。由于游戏规则中，当玩家手上只余下一张牌时，必须喊出"uno"，因而得名。Uno是西班牙语和意大利语中“1”的意思。

#  总体设计

## 老师给的ppt上面的要求：

### 基本目标

- 友好的图形用户界面
- 支持2种扑克牌游戏：改为支持多种uno游戏模式
- 支持四玩家参与游戏：改为支持 2 - 8人游戏

### 高级目标

- 支持单人游戏，不足人数换成AI
- 支持不同玩家局域网内联机参与游戏
